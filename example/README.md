# Kotchan Example

## Run

### JVM
Run the application
    
    $ ./gradlew run
    
### iOS
1. Build on ios project directory

        $ cd ios
        $ ./gradlew build

2. Open xcode project (`ios/src/iosMain/xcode/kotchan-example.xcodeproj`)

3. Run the application on xcode

### Android
Not supported yet
